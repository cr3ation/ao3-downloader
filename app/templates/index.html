Author

Searches a user's works page. Use the exact AO3 username, e.g. missyuki1990. You can find it in any work URL: archiveofourown.org/users/username.

Tag / Fandom

Searches an AO3 tag page. Tags cover fandoms, relationships, characters and freeform tags — for example:

  • Fandom: Harry Potter - J. K. Rowling (often just Harry Potter works too)
  • Relationship: Draco Malfoy/Harry Potter
  • Freeform: Enemies to Lovers, Fluff

If the exact tag doesn't exist, the app automatically falls back to AO3's general work search.

Advanced queries

In the general search (used as tag fallback) AO3 supports operators:

  • "exact phrase" — match the exact words
  • Harry AND Draco, Sith OR Jedi — boolean logic
  • -wordToExclude — exclude a term
  • wolf* — wildcard

The Filters above (complete only, word count, excluded tags) apply to both Author and Tag searches.

Searches are rate-limited to be polite to AO3 — multi-page searches can take a while.
